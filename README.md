🐾 Abrigo Patas Felizes
Este é um sistema de gerenciamento de abrigo de animais desenvolvido em Java. O projeto foi criado como parte de uma atividade prática para exercitar conceitos de Programação Orientada a Objetos (POO) e manipulação de dados via console.

📋 Sobre o Projeto
O PatasFelizes permite que um usuário (administrador do abrigo) realize o cadastro organizado de diferentes tipos de animais, armazene suas características principais e liste todos os registros salvos em um "Abrigo".

Funcionalidades:
Menu Interativo: Navegação via terminal utilizando a classe Scanner.

Registro de Cachorros: Cadastro de espécie, nome, idade e raça.

Registro de Gatos: (Funcionalidade preparada no menu).

Registro de Outros Animais: Flexibilidade para diferentes espécies.

Listagem Geral: Visualização de todos os animais cadastrados no abrigo.

🛠️ Tecnologias e Conceitos Aplicados
Linguagem: Java

Entrada de Dados: java.util.Scanner

Estruturas de Repetição/Decisão: while(true) e switch/case.

Orientação a Objetos:

Classes e Objetos: Uso das classes Cachorro, Gato e Abrigo.

Encapsulamento: Uso de métodos set (ex: c.setNome()) para atribuir valores.

Coleções: (Provável uso de List ou Array dentro da classe Abrigo).

📂 Estrutura de Pastas
Plaintext
src/
 ├── dados/      # Contém as classes de modelo (Cachorro, Gato, Abrigo e OutrosAnimais)
 └── tela/       # Contém a classe Principal (Interface com o usuário)
🚀 Como executar

Clone ou baixe os arquivos do projeto.

No seu terminal ou IDE, compile e execute a classe Principal.java.

Bash
# Exemplo via terminal
javac tela/Principal.java
java tela.Principal
